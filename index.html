Snitch

JVM monitoring finally made painless.

Clever marketing tagline here

Built on open protocols

No custom tools or protocols here. Snitch is built to integrate seamlessly into a modern HTTP stack and exposes its data as JSON over HTTP. Data is accessible by any web browser or through an HTTP client written in any language.

Trivial to use

Monitoring is one of those difficult tasks that needs to permeate across an entire system. Snitch recognizes this and makes it as simple as possible for developers to integrate it into existing systems.

Monitor anything

Snitch doesn't put any restrictions on what can and can't be monitored.

Fast

Snitch is designed to be fast and to have minimal impact on the performance of your system.
